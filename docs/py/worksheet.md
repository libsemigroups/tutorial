# Worksheet

This page contains some exercises for computing some things using `libsemigroups_pybind11`.

1. Let $J_n$ denote the Jones monoid of degree $n$, and
   ![An element of the Jones monoid](../images/bipartition.png)
   be a bipartition of degree 14.
   1. Find a collection of bipartitions $\{x_1, x_2, \dots, x_j\}$ such that
      the tensor product $x_1 \otimes x_2 \otimes \dots \otimes x_j$ is equal to
      $x$. What is the largest possible value of $j$? For the definition of the
      tensor product, see the documentation of the function
      `TensorBipartition`.
   2. How many elements does $J_{14}$ contain?
   3. How many idempotent elements does $J_{14}$ contain?
   4. How many idempotent elements does the Green's
      $\mathscr{D}$-class of $x$ in $J_{14}$ contain?
   5. How many idempotent elements $e$, for which $7$ is in the
      domain of $e$, does the Green's $\mathscr{D}$-class of $x$ in
      $J_{14}$ contain?
   6. How many irreducible idempotent elements $\hat{e}$, for
      which $7$ is in the domain of $\hat{e}$, does the Green's
      $\mathscr{D}$-class of $x$ in $J_{14}$ contain?
